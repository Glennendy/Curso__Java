Aula 10-Tipos primitivos-Convenção de variáveis

Tem 8 tipos primitivos, elas são também, palavras reservadas.

Todas são escritas com letra minuscula.

Os tipos são: int, double, float, char, byte, short, long, boolean

Para criar uma variável, primeiro coloca o tipo dela, depois da um nome.

Ob.: Quando for dar o nome, diferente das classes em que a primeira letra é maiuscula,
na variável, a primeira letra é minúscula, caso o nome tenha palavras compostas, a primeira 
letra da primeira palavra é minúscula e as outras palavras, a primeira é maiúscula.
Exemplo: idadeDoPai

Para iniciar uma variável, usa o sinal de igual.

int idade = 10;

Para imprimir o valor da variável no console, coloca o nome da variável sem aspas duplas.

System.out.println(idade);

Para adicionar texto, coloca aspas duplas e o sinal de +

("A idade é "+idade);
